# Era

Era is a virtual wardrobe app that helps you dress from clothes you already own.
Ovi, the AI stylist, lives in a floating button and gives weather-aware daily outfit suggestions.
Built as a pnpm + Turborepo monorepo: `apps/` (web, mobile) and `packages/` (core, tokens, db, configs).
Get started with `pnpm install`, then `turbo run lint` to check the workspace.
